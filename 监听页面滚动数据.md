## 监听页面滚动，顶部导航栏固定定位
>methods里面的代码
``` vue
       /* 滚动监听定位 */
        promScroll () {
            var scrollTop = window.pageYOffset || document.documentElement.scrollTop || document.body.scrollTop
            /* 下面的类目是你要监听的元素类名 */
            var offsetTop = document.querySelector('类名').offsetTop
            if (scrollTop > offsetTop) {
                这里可以写你要改变的事件或者是状态
            } else {
                这里可以写你要改变的事件或者是状态
            }
        },
```
