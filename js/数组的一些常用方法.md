### 数组取值：最大最小
    ```js
        /* 取最大值： */ Math.max.apply(Math, array.map(function(o) {return o.value}))
        /* 取最小值： */ Math.min.apply(Math, array.map(function(o) {return o.value}))
        //说明：array是我们上面定义的数组，o.value就是要比较的字段。
    ```
